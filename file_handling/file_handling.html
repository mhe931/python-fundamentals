File Handling

The open() Function

The open() function takes two parameters; filename and mode.

There are four different methods (modes) for opening a file:

  • "r" - Read - Default value. Opens a file for reading, error if the file does not exist
  • "a" - Append - Opens a file for appending, creates the file if it does not exist
  • "w" - Write - Opens a file for writing, creates the file if it does not exist
  • "x" - Create - Creates the specified file, returns an error if the file exists

In addition you can specify if the file should be handled as binary or text mode

  • "t" - Text - Default value. Text mode
  • "b" - Binary - Binary mode (e.g. images)

ðŸ’¡ Tips and Tricks

  • Always use with statement to automatically close files
  • Use pathlib.Path for better file path handling
  • Remember to handle exceptions when working with files
  • Use 'rb' mode when working with binary files like images

# Best practice with context manager
with open('file.txt', 'r') as f:
    content = f.read()
# File automatically closes

Read Files

Read Files

The open() function returns a file object, which has a read() method for reading the content of the file:


f = open("demofile.txt", "r")
print(f.read())

Read Parts of a File

By default the read() method returns the whole text, but you can also specify how many characters you want to return:


f = open("demofile.txt", "r")
print(f.read(5))

Read Lines

You can return one line by using the readline() method:


f = open("demofile.txt", "r")
print(f.readline())

Loop Through the File

By looping through the lines of the file, you can read the whole file, line by line:


f = open("demofile.txt", "r")
for x in f:
  print(x)

ðŸ’¡ Tips and Tricks

  • Use readlines() to get a list of all lines at once
  • For large files, iterate line by line to save memory
  • Use strip() to remove trailing newlines
  • Remember: read() reads entire file into memory

# Memory efficient reading
with open('large_file.txt', 'r') as f:
    for line in f:
        process(line.strip())

ðŸŽ‰ Fun Facts

  • readline() keeps the newline character \n at the end!
  • Reading a file twice requires calling seek(0) to reset the pointer
  • Empty lines in a file are read as "\n", not empty strings
  • File iteration is lazy - it doesn't load the whole file into memory

Write/Create Files

Write to an Existing File

To write to an existing file, you must add a parameter to the open() function:

  • "a" - Append - will append to the end of the file
  • "w" - Write - will overwrite any existing content

f = open("demofile2.txt", "a")
f.write("Now the file has more content!")
f.close()

#open and read the file after the appending:
f = open("demofile2.txt", "r")
print(f.read())

Create a New File

To create a new file in Python, use the open() method, with one of the following parameters:

  • "x" - Create - will create a file, returns an error if the file exist
  • "a" - Append - will create a file if the specified file does not exist
  • "w" - Write - will create a file if the specified file does not exist

f = open("myfile.txt", "x")

ðŸ’¡ Tips and Tricks

  • Use "w" carefully - it erases existing content!
  • writelines() writes a list of strings to a file
  • Use "a+" to append and read simultaneously
  • Always call flush() for critical writes

lines = ['Line 1\n', 'Line 2\n', 'Line 3\n']
with open('output.txt', 'w') as f:
    f.writelines(lines)

ðŸŽ‰ Fun Facts

  • Python buffers writes for performance - data may not write immediately!
  • Mode "x" is exclusive creation - fails if file exists (since Python 3.3)
  • write() returns the number of characters written
  • You can write to sys.stdout like a file - it's what print() uses!

Delete Files

Delete a File

To delete a file, you must import the os module, and run its os.remove() function:


import os
os.remove("demofile.txt")

Check if File exist

To avoid getting an error, you might want to check if the file exists before you try to delete it:


import os
if os.path.exists("demofile.txt"):
  os.remove("demofile.txt")
else:
  print("The file does not exist")

Delete Folder

To delete an entire folder, use the os.rmdir() method:


import os
os.rmdir("myfolder")

Note: You can only remove empty folders.

ðŸ’¡ Tips and Tricks

  • Use shutil.rmtree() to delete non-empty directories
  • pathlib.Path.unlink() is the modern way to delete files
  • Always check file existence before deletion to avoid errors
  • Use try-except blocks for safer file deletion

from pathlib import Path
file = Path('myfile.txt')
if file.exists():
    file.unlink()

ðŸŽ‰ Fun Facts

  • Deleted files aren't immediately erased - the space is just marked as available!
  • os.remove() and os.unlink() are identical functions
  • On Windows, you can't delete a file that's currently open
  • The send2trash module moves files to recycling bin instead of permanent deletion
