Built-in Modules

Built-in Modules

Python has a set of built-in modules that you can use whenever you like.

You can get a list of all built-in modules by running the following in a Python interpreter:


help('modules')

Random Module

Generate Random Numbers

Python does not have a random() function to make a random number, but Python has a built-in module called random that can be used to make random numbers:


import random

print(random.randrange(1, 10))

Random Methods

The random module has a set of methods:

  • seed() - Initializes the random number generator
  • getstate() - Returns the current internal state of the random number generator
  • setstate() - Restores the internal state of the random number generator
  • randrange() - Returns a random number between the given range
  • randint() - Returns a random number between the given range (inclusive)
  • choice() - Returns a random element from the specified sequence
  • choices() - Returns a list with a random selection from the specified sequence
  • shuffle() - Takes a sequence and returns the sequence in a random order
  • sample() - Returns a given sample of a sequence

Requests Module

Make an HTTP Request

The requests module allows you to send HTTP requests using Python.

The requests module is not built-in to Python. You have to install it first:

pip install requests

Once you have installed the requests module, you can import it and start making requests:


import requests

x = requests.get('https://w3schools.com/python/demopage.htm')

print(x.status_code)

Statistics Module

Statistical Functions

The statistics module provides functions for calculating mathematical statistics of numeric (Real-valued) data.


import statistics

x = [1, 2, 3, 4, 5, 6, 7, 8, 9]
print(statistics.mean(x))
print(statistics.median(x))
print(statistics.mode(x))

Math Module

Mathematical Functions

The math module provides access to the mathematical functions defined by the C standard.


import math

print(math.sqrt(64))
print(math.ceil(1.4))
print(math.floor(1.4))
print(math.pi)

cMath Module

Complex Mathematical Functions

The cmath module provides access to mathematical functions for complex numbers.


import cmath

print(cmath.sqrt(-1))
print(cmath.log(10, 2))
