üìä Data Structures

Data structures are specialized formats for organizing, processing, and storing data. Understanding these structures is crucial for writing efficient and scalable code.

Arrays & Lists

Dynamic collections that store elements sequentially. Python lists are versatile and support various operations.

Linked Lists

Linear data structures where elements are connected using pointers. Efficient for insertions and deletions.

Stacks

LIFO (Last In First Out) structure. Perfect for undo operations, expression evaluation, and backtracking.

Queues

FIFO (First In First Out) structure. Used in scheduling, breadth-first search, and buffering.

Trees

Hierarchical structures with parent-child relationships. Binary trees, BST, AVL trees, and more.

Graphs

Networks of nodes connected by edges. Essential for social networks, maps, and dependency resolution.

Hash Tables

Key-value pairs with O(1) average lookup time. Python dictionaries are implemented as hash tables.

Heaps

Complete binary trees used for priority queues and efficient min/max operations.

Example: Implementing a Stack

class Stack:
    def __init__(self):
        self.items = []

    def push(self, item):
        self.items.append(item)

    def pop(self):
        if not self.is_empty():
            return self.items.pop()

    def peek(self):
        if not self.is_empty():
            return self.items[-1]

    def is_empty(self):
        return len(self.items) == 0

# Usage
stack = Stack()
stack.push(1)
stack.push(2)
print(stack.pop()) # Output: 2

‚ö° Algorithms

Algorithms are step-by-step procedures for solving problems. Learning algorithms helps you write more efficient code and tackle complex challenges.

Sorting

Bubble Sort, Quick Sort, Merge Sort, Heap Sort. Arrange data in a specific order.

Searching

Linear Search, Binary Search. Find elements efficiently in data structures.

Recursion

Functions calling themselves. Essential for tree traversal and divide-and-conquer approaches.

Dynamic Programming

Optimize recursive solutions by storing results. Solve complex problems efficiently.

Greedy Algorithms

Make locally optimal choices. Used in scheduling, Huffman coding, and graph problems.

Graph Algorithms

BFS, DFS, Dijkstra's, A*. Navigate and analyze graph structures.

Backtracking

Explore all possibilities by trying and undoing choices. Solve puzzles and constraint problems.

Divide & Conquer

Break problems into smaller subproblems. Merge Sort and Quick Sort use this approach.

Example: Binary Search

def binary_search(arr, target):
    left, right = 0, len(arr) - 1

    while left <= right:
        mid = (left + right) // 2

        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1

# Usage
numbers = [1, 3, 5, 7, 9, 11, 13]
result = binary_search(numbers, 7)
print(f"Found at index: {result}") # Output: Found at index: 3
