Python Numbers

Working with numeric types in Python.

Numeric Types

There are three numeric types in Python:

  • int
  • float
  • complex

x = 1    # int
y = 2.8  # float
z = 1j   # complex
Speaker notes: Explain the difference between these types. Integers are whole numbers, floats have decimals, and complex numbers have a real and imaginary part.

Type Conversion

You can convert from one type to another with the int(), float(), and complex() methods:


x = 1    # int
y = 2.8  # float
z = 1j   # complex

#convert from int to float:
a = float(x)

#convert from float to int:
b = int(y)

#convert from int to complex:
c = complex(x)

Random Number

Python does not have a random() function to make a random number, but Python has a built-in module called random that can be used to make random numbers:


import random

print(random.randrange(1, 10))
Speaker notes: Explain that the `randrange` function returns a number between the given range.
