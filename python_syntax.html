Execute Python Syntax

Python syntax can be executed by writing directly in the Command Line or by creating a python file (.py) and running it in the Command Line.

Python Indentation

Indentation refers to the spaces at the beginning of a code line. In other programming languages the indentation in code is for readability only, but in Python, it is very important.

Indentation Matters

Python uses indentation to indicate a block of code.


if 5 > 2:
  print("Five is greater than two!")
Speaker notes: Emphasize that the number of spaces is variable, but it has to be consistent. The convention is four spaces.
