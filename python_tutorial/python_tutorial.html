Python Intro

What is Python?

Python is a popular programming language. Created by Guido van Rossum, released in 1991.

Used for:

  • Web development (server-side)
  • Software development
  • Mathematics
  • System scripting

What can Python do?

  • Create web applications.
  • Create workflows.
  • Connect to database systems, read/modify files.
  • Handle big data and complex math.
  • Used for rapid prototyping.

Why Python?

  • Works on different platforms (Windows, Mac, Linux, etc).
  • Simple syntax, similar to English.
  • Fewer lines of code than other languages.
  • Interpreter system for quick prototyping.
  • Supports procedural, object-oriented, and functional programming.

Python Get Started

Python Install

Many PCs and Macs have Python installed. Check by running:

python --version

Python Quickstart

Python is interpreted. Run code directly in the command line:

python
>>> print("Hello, World!")

Or run a Python file (myfile.py):

python myfile.py

Python Syntax

Execute Python Syntax

Execute Python by writing directly in the Command Line or by creating a python file (.py) and running it.

Python Indentation

Indentation: spaces at the beginning of a code line. In Python, it is very important, unlike other languages where it's for readability only.

Indentation Matters

Python uses indentation to indicate a block of code.


if 5 > 2:
  print("Five is greater than two!")

Python Statements

A statement is an instruction the Python interpreter executes. E.g., print and assignment statements.


# Single statement
print("Hello, World!")

# Another single statement
x = 5

Multi-line statements use the line continuation character (\\):


x = 1 + 2 + 3 + \
    4 + 5 + 6

Python Output

Python Output

The Python print() function outputs variables.


x = "Python is awesome"
print(x)

Output Variables

Output multiple variables in print(), separated by a comma:


x = "Python"
y = "is"
z = "awesome"
print(x, y, z)

Use the + operator to output multiple variables:


x = "Python "
y = "is "
z = "awesome"
print(x + y + z)

Python Comments

Comments

Comments explain code, improve readability, and prevent execution during testing.

Start with #; Python ignores them:


#This is a comment
print("Hello, World!")

Multi Line Comments

Python doesn't have a specific multi-line comment syntax. Use # for each line:


#This is a comment
#written in
#more than just one line
print("Hello, World!")

Or use a multiline string (not strictly comments, but often used as such):


'''
This is a comment
written in
more than just one line
'''
print("Hello, World!")

Python Variables

Creating Variables

Variables are containers for data values. Python has no command for declaring a variable. Created when you first assign a value.


# x is of type int
x = 5
# y is of type str
y = "John"
print(x)
print(y)

Variable Names

Rules for Python variables:

  • Starts with letter or underscore.
  • Cannot start with a number.
  • Only alpha-numeric characters and underscores.
  • Case-sensitive (age, Age, AGE are different).

Assign Multiple Values

Assign values to multiple variables in one line:


x, y, z = "Orange", "Banana", "Cherry"

Assign the same value to multiple variables:


x = y = z = "Orange"

Python Data Types

Built-in Data Types

Data type is important. Variables store data of different types. Python has these built-in categories:

  • Text: str
  • Numeric: int, float, complex
  • Sequence: list, tuple, range
  • Mapping: dict
  • Set: set, frozenset
  • Boolean: bool
  • Binary: bytes, bytearray, memoryview

Getting the Data Type

Use the type() function to get an object's data type:


x = 5
print(type(x))

Python Numbers

Numeric Types

Python has three numeric types:

  • int (whole numbers)
  • float (numbers with decimals)
  • complex (numbers with an imaginary part)

x = 1    # int
y = 2.8  # float
z = 1j   # complex

Type Conversion

Convert between types with int(), float(), and complex() methods:


x = 1    # int
y = 2.8  # float

# Convert int to float:
a = float(x)

# Convert float to int:
b = int(y)

Python Casting

Specify a Variable Type

Use casting to specify a variable's type. Python uses classes to define data types.

Casting uses constructor functions:

  • int(): from int, float (removes decimals), or string (whole number).
  • float(): from int, float, or string (float/int).
  • str(): from various data types.

Casting Examples


x = int(1)   # x will be 1
y = int(2.8) # y will be 2
z = int("3") # z will be 3

x = float(1)     # x will be 1.0
y = float(2.8)   # y will be 2.8
z = float("3")   # z will be 3.0
w = float("4.2") # w will be 4.2

x = str("s1") # x will be 's1'
y = str(2)    # y will be '2'
z = str(3.0)  # z will be '3.0'

Python Strings

Strings

Strings are surrounded by single or double quotation marks.


print("Hello")
print('Hello')

String as Arrays

Strings are arrays of bytes (Unicode characters). Python has no character type; a single character is a string of length 1.


a = "Hello, World!"
print(a[1]) # Output: e

String Methods

Python has many built-in methods for strings:


a = " Hello, World! "
print(a.strip())   # Removes whitespace: "Hello, World!"
print(len(a))      # Length: 15
print(a.lower())   # Lowercase: " hello, world! "
print(a.upper())   # Uppercase: " HELLO, WORLD! "
print(a.replace("H", "J")) # Replace: " Jello, World! "
print(a.split(",")) # Splits: [' Hello', ' World! ']

Python Booleans

Booleans

Expressions evaluate to True or False. Python returns the Boolean answer when comparing values.


print(10 > 9)  # True
print(10 == 9) # False
print(10 < 9)  # False

Most Values are True

Almost any value with content is True. Examples:

  • Any string (except empty strings).
  • Any number (except 0).
  • Any list, tuple, set, or dictionary (except empty ones).

Some Values are False

The following will return False:


bool(False)
bool(None)
bool(0)
bool("")
bool(())
bool([])
bool({})

Python Operators

Operators

Operators perform operations on variables and values. Python groups operators:

  • Arithmetic
  • Assignment
  • Comparison
  • Logical
  • Identity
  • Membership
  • Bitwise

Arithmetic Operators

Arithmetic operators perform mathematical operations with numeric values:


print(10 + 5)  # Addition
print(10 - 5)  # Subtraction
print(10 * 5)  # Multiplication
print(10 / 5)  # Division
print(10 % 5)  # Modulus
print(10 ** 5) # Exponentiation
print(10 // 5) # Floor division

Logical Operators

Logical operators combine conditional statements:


print(10 > 9 and 10 > 8) # True
print(10 > 9 or 10 < 8)  # True
print(not(10 > 9 and 10 > 8)) # False

Python Lists

Lists

Lists store multiple items in a single variable. They are ordered, changeable, and allow duplicate values.


thislist = ["apple", "banana", "cherry"]
print(thislist)

List Items

List items are ordered, changeable, and allow duplicate values. They are indexed (first item: [0], second: [1], etc.).

List Methods

Python has many built-in methods for lists:


thislist = ["apple", "banana", "cherry"]
thislist.append("orange") # Add item
print(thislist)

thislist.remove("banana") # Remove item
print(thislist)

print(len(thislist)) # Length

Python Tuples

Tuples

Tuples store multiple items in a single variable. They are ordered and unchangeable. Written with round brackets.


thistuple = ("apple", "banana", "cherry")
print(thistuple)

Tuple Items

Tuple items are ordered, unchangeable, and allow duplicate values. They are indexed (first item: [0], second: [1], etc.).

Tuple Methods

Python has two built-in methods for tuples:


thistuple = ("apple", "banana", "cherry")
print(thistuple.count("apple"))  # Count occurrences
print(thistuple.index("cherry")) # Find index

Note: Few methods due to unchangeable nature.

Python Sets

Sets

Sets store multiple items in a single variable. They are unordered and unindexed. Written with curly brackets.


thisset = {"apple", "banana", "cherry"}
print(thisset)

Set Items

Set items are unordered, unchangeable, and do not allow duplicate values.

Set Methods

Python has built-in methods for sets:


thisset = {"apple", "banana", "cherry"}
thisset.add("orange")    # Add item
print(thisset)

thisset.remove("banana") # Remove item
print(thisset)

print(len(thisset))      # Length

Python Dictionaries

Dictionaries

Dictionaries store data in key:value pairs. They are ordered*, changeable, and do not allow duplicates.


thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
print(thisdict)
Speaker notes: As of Python version 3.7, dictionaries are ordered. In Python 3.6 and earlier, dictionaries are unordered.

Dictionary Items

Items are key:value pairs, referred to by key name.


thisdict = {"brand": "Ford", "model": "Mustang"}
print(thisdict["brand"]) # Output: Ford

Dictionary Methods

Python has built-in methods for dictionaries:


thisdict = {"brand": "Ford", "model": "Mustang"}
thisdict["color"] = "red" # Add item
print(thisdict)

thisdict.pop("model") # Remove item
print(thisdict)

print(len(thisdict))  # Length

Python If...Else

If...Else

Python supports logical conditions from mathematics:

  • Equals: a == b
  • Not Equals: a != b
  • Less than: a < b
  • Less than or equal to: a <= b
  • Greater than: a > b
  • Greater than or equal to: a >= b

Used in "if statements" and loops.

If Statement

An "if statement" uses the if keyword.


a = 33
b = 200
if b > a:
  print("b is greater than a")

Elif

The elif keyword: "if previous conditions were false, try this condition".


a = 33
b = 33
if b > a:
  print("b is greater than a")
elif a == b:
  print("a and b are equal")

Else

The else keyword catches anything not caught by preceding conditions.


a = 200
b = 33
if b > a:
  print("b is greater than a")
elif a == b:
  print("a and b are equal")
else:
  print("a is greater than b")

Python Match

Match Statement

A match statement compares an expression's value to patterns in case blocks.

Similar to switch statements in other languages.


def http_status(status):
    match status:
        case 400: return "Bad request"
        case 404: return "Not found"
        case 418: return "I'm a teapot"
        case _:   return "Something's wrong"

Python While Loops

While Loops

The while loop iterates a code block as long as the test expression (condition) is true.


i = 1
while i < 6:
  print(i)
  i += 1

The break Statement

The break statement stops the loop even if the while condition is true:


i = 1
while i < 6:
  print(i)
  if i == 3:
    break
  i += 1

The continue Statement

The continue statement stops the current iteration and continues with the next:


i = 0
while i < 6:
  i += 1
  if i == 3:
    continue
  print(i)

Python For Loops

For Loops

A for loop iterates over a sequence (list, tuple, dictionary, set, or string).


fruits = ["apple", "banana", "cherry"]
for x in fruits:
  print(x)

The range() Function

The range() function returns a sequence of numbers. Starts from 0 by default, increments by 1 (by default), and ends at a specified number.


for x in range(6):
  print(x)

Nested Loops

A nested loop is a loop inside a loop. The inner loop executes once for each outer loop iteration.


adj = ["red", "big", "tasty"]
fruits = ["apple", "banana", "cherry"]

for x in adj:
  for y in fruits:
    print(x, y)

Functions

A function is a block of code that only runs when called. Pass parameters into it. Can return data.

Python OOP

Object-Oriented Programming

Python is an object-oriented programming language.

Almost everything in Python is an object with properties and methods. A Class is like an object constructor, a "blueprint" for creating objects.

Python Classes/Objects

Create a Class

Use the keyword class to create a class:


class MyClass:
  x = 5

Create Object

Use the class MyClass to create objects:


p1 = MyClass()
print(p1.x)

The __init__() Function

The built-in __init__() function is always executed when the class is initiated.

Use it to assign values to object properties or perform necessary operations during object creation:


class Person:
  def __init__(self, name, age):
    self.name = name
    self.age = age

p1 = Person("John", 36)

print(p1.name)
print(p1.age)

Python Inheritance

Python Inheritance

Inheritance allows a class to inherit methods and properties from another class.

  • Parent class (base class): Inherited from.
  • Child class (derived class): Inherits from parent.

Create a Parent Class

Any class can be a parent class:


class Person:
  def __init__(self, fname, lname):
    self.firstname = fname
    self.lastname = lname

  def printname(self):
    print(self.firstname, self.lastname)

# Example:
x = Person("John", "Doe")
x.printname()

Create a Child Class

Pass the parent class as a parameter:


class Student(Person):
  pass

Add the __init__() Function

Adding __init__() to child class stops inheriting parent's __init__().

To keep parent's __init__(), call it explicitly:


class Student(Person):
  def __init__(self, fname, lname):
    Person.__init__(self, fname, lname)

Use the super() Function

The super() function makes the child class inherit all methods and properties from its parent:


class Student(Person):
  def __init__(self, fname, lname):
    super().__init__(fname, lname)

Python Iterators

Python Iterators

An iterator is an object with a countable number of values, allowing traversal through all values.

Implements __iter__() and __next__() methods.

Iterator vs Iterable

Lists, tuples, dictionaries, sets are iterable objects. They have an iter() method to get an iterator:


mytuple = ("apple", "banana", "cherry")
myit = iter(mytuple)

print(next(myit))
print(next(myit))
print(next(myit))

Create an Iterator

Implement __iter__() and __next__() in your object/class.

  • __iter__(): Returns the iterator object itself.
  • __next__(): Returns the next item in the sequence.

class MyNumbers:
  def __iter__(self):
    self.a = 1
    return self

  def __next__(self):
    x = self.a
    self.a += 1
    return x

myclass = MyNumbers()
myiter = iter(myclass)

print(next(myiter))
print(next(myiter))
print(next(myiter))
print(next(myiter))
print(next(myiter))

Python Polymorphism

What is Polymorphism?

Polymorphism means "many forms". In programming, it refers to methods/functions/operators with the same name that can be executed on many objects or classes.

Polymorphism in Class Methods

Multiple classes can have the same method name:


class Car:
  def __init__(self, brand, model):
    self.brand = brand
    self.model = model
  def move(self): print("Drive!")

class Boat:
  def __init__(self, brand, model):
    self.brand = brand
    self.model = model
  def move(self): print("Sail!")

class Plane:
  def __init__(self, brand, model):
    self.brand = brand
    self.model = model
  def move(self): print("Fly!")

car1 = Car("Ford", "Mustang")
boat1 = Boat("Ibiza", "Touring 20")
plane1 = Plane("Boeing", "747")

for x in (car1, boat1, plane1):
  x.move()

Python Dates

Python doesn't have a built-in date type. Import the datetime module to work with date objects.


import datetime

Python Math

Built-in Math Functions

min() and max(): find lowest/highest value.


x = min(5, 10, 25)
y = max(5, 10, 25)
print(x, y)

abs(): returns absolute (positive) value.


x = abs(-7.25)

x = pow(4, 3)

import math

Python JSON

What is JSON?

JSON is a syntax for storing and exchanging data. It's text, written with JavaScript object notation.

Parse JSON (JSON to Python)

Use json.loads() to parse a JSON string into a Python dictionary:


import json

x =  '{ "name":"John", "age":30, "city":"New York"}'
y = json.loads(x)
print(y["age"]) # Output: 30

Convert Python to JSON

Use json.dumps() to convert a Python object into a JSON string:


import json

x = {"name": "John", "age": 30, "city": "New York"}
y = json.dumps(x)
print(y)

Format the Result

json.dumps() has parameters for readability (indentations, line breaks):


import json

x = {"name": "John", "age": 30, "city": "New York"}
print(json.dumps(x, indent=4))

Python RegEx

What is a Regular Expression?

A RegEx (Regular Expression) is a sequence of characters forming a search pattern.

Used to check if a string contains the specified pattern.

RegEx Module

Python's built-in re package works with Regular Expressions. Import it:


import re

RegEx Functions

The re module offers functions to search for matches:

  • findall: Returns a list of all matches.
  • search: Returns a Match object if a match is found.
  • split: Returns a list where the string is split at each match.
  • sub: Replaces matches with a string.

Python Try...Except

Exception Handling

The try block tests code for errors.

The except block handles the error.

The finally block executes regardless of try/except outcome.


try:
  print(x)
except:
  print("An exception occurred")

Many Exceptions

Define multiple except blocks for specific error types:


try:
  print(x)
except NameError:
  print("Variable x is not defined")
except:
  print("Something else went wrong")

Finally Block

The finally block executes whether an error occurs or not.


try:
  print(x)
except:
  print("Something went wrong")
finally:
  print("The 'try except' is finished")
